Natours | Responsive web app
Tours selling web app, the user can perform the following activities:

1. create account
2. review tours
3. book tours
4. compleate online payment
5. recive auto sent emails
6. add review
7. rate tours
   Technologies used:
8. CSS, HTML and javascript.
9. Node.js
10. mongo DB
11. mongoose
12. express
13. mail grid
14. stripe
